# NeoFeed 双重宇宙与官网重构更新日志 (2026-01-28)

## 🌌 1. 双重宇宙 (Dual Universe) 核心升级

### 快思考宇宙 (Fast-Thinking Universe)
*   **视觉重构**：采用了 Deep Rock Galactic (深岩银河) 风格，强调混乱、宏伟与数据碎片感。
*   **核心元素**：
    *   **紫色深邃内核**：更小、更精致的脉动八面体作为能量源。
    *   **Manga 风格晶体**：非规则几何碎片，带有硬朗的赛璐璐边缘线 (Outlines) 和运动拖尾 (Trail)。
    *   **白色流光粒子**：均匀分布在空间中，从左上往右下平滑流动，营造空间穿梭感。
    *   **空间分割线**：冷冽、粗细不一的穿刺直线，增强了空间的几何结构感。

### 慢思考宇宙 (Slow-Thinking Universe)
*   **视觉重构**：强调秩序、整洁与知识结晶的优美。
*   **核心元素**：
    *   **极光星云 (Galaxy Aurora)**：引入了非规则、纤细的极光带，替代了生硬的圆球氛围。
    *   **拟真星球地表**：星球表面具备多层噪声地形、动态城市灯光（暗面）和菲涅尔大气边缘光。
    *   **物理动效**：引入非线性动力学，相机带有微弱的“呼吸感”漂浮，模拟星系引力。
*   **交互升级**：
    *   **星系聚焦**：点击星系核心，相机平滑拉近并切换至该星系视角。
    *   **2D 俯视模式**：一键切换至上帝视角，星轨清晰可见，便于全局审视知识版图。
    *   **视角重置**：智能重置按钮，根据当前模式平滑回归初始机位。

## 🛠️ 2. 交互与功能深度优化

### 知识处理流 (Knowledge Processing)
*   **双栏详情模态框**：点击知识点弹出左读右写的双栏界面（左侧内容，右侧笔记/标签/权重）。
*   **AI 格式化引擎**：
    *   **Markdown 重构**：AI 自动将抓取的乱码或平铺文本重写为结构清晰、带小标题和列表的 Markdown。
    *   **容错解析**：针对长文处理，开发了三重容错解析逻辑（JSON -> 正则提取 -> 全量保底），确保内容不丢失。
*   **旧数据翻新脚本**：创建了 `/api/admin/reformat-feeds` 接口，支持分批、按时间顺序重构所有历史数据。

### 界面交互 (UI/UX)
*   **星际终端联动**：点击内容时，左侧搜索列表自动平滑收起；关闭详情时自动恢复。
*   **HUD 增强**：慢宇宙下方新增统计栏，实时展示知识总量、核心领域及同步时间。

## 🌐 3. 官网 (Landing Page) 视觉重塑

### 风格定义
*   **Matrix x Citypop**：融合了《黑客帝国》的赛博黑绿风格与 Citypop 的复古未来主义渐变色。
*   **赛璐璐渲染**：整体视觉采用 90 年代日本动画质感，光影硬朗，高光锐利。

### 响应式与大屏适配
*   **超宽屏优化**：针对 2K/4K 屏幕，将主容器扩展至 1600px，并实现了全页面的等比例缩放。
*   **Bento 布局**：重新设计了“万物接入”与“理解万物”两个核心展示模块，采用对角线构图与非对称美学。
*   **3D 监视器**：Hero 区域加入可交互的金属质感 3D 监视器模型。

---

## 🐞 4. 关键修复与系统稳定性
*   **Auth 状态同步**：修复了 Middleware 拦截 API 导致的 401 错误。
*   **Hydration 修复**：解决了 Next.js 常见的服务端与客户端渲染不一致导致的刷新问题。
*   **AI 配置联动**：重构脚本现在能正确读取用户个人配置的 API Key，解决了重构时的权限拦截。
