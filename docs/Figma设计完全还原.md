## 🔧 技术实现

### 框架 & 库
- **Next.js 16** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式系统
- **Framer Motion** - 动画库
- **shadcn/ui** - UI 组件库
- **Lucide React** - 图标库

### 关键特性
1. **服务端组件 + 客户端组件**
   - 使用 `'use client'` 声明客户端组件
   - 支持交互和状态管理

2. **响应式设计**
   - 移动端：1列
   - 平板：2列
   - 桌面：4列

3. **后端集成**
   - 调用 FastAPI 的 `/api/items` 端点
   - POST 请求保存内容
   - 错误处理和加载状态

4. **路由系统**
   - Next.js App Router
   - 文件系统路由
   - `useRouter` 编程式导航
