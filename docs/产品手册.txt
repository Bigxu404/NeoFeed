# NeoFeed 产品使用手册 🧠

欢迎使用 NeoFeed，这是一个专为深度信息消费者打造的“个人情报站”。通过 AI 的力量，我们将散乱的网页链接和冗长的 RSS 订阅转化为极度精炼、可消费的结构化情报。

---

## 核心功能简介

### 1. 手动捕捉与深度总结
当你在网络上发现一篇深度好文、研究报告或视频时，只需将其 URL 提交给 NeoFeed：
- **全量抓取**：自动解析网页正文，去除广告噪音。
- **AI 深度阅读**：利用大模型对内容进行结构化总结（包含核心观点、背景、结论）。
- **永久存档**：所有总结内容都将永久存储在你的个人“矩阵 FEED”中，随时供后续查阅和导出周报。

### 2. RSS 信号拦截与智能推送
无需再盯着 RSS 阅读器看几百条标题。NeoFeed 充当了你的“第一读者”：
- **全天候监测**：系统按照你设定的频率（每天/每周）自动扫描全球学术期刊、科技媒体。
- **结构化阅读**：AI 自动阅读每篇文章，并提炼为“一句话总结”、“研究方法”和“研究结果”。
- **精美周报**：按期刊分类、经过视觉优化的“纽约客”风格周报将准时发送至你的邮箱。

## 配置指南

### 第一步：注入“神经核心”（AI 配置）
在使用任何 AI 功能前，你需要为 NeoFeed 提供“思考动力”：
1. 进入 **[设置页] -> [神经核心]**。
2. 配置 **API Key**：支持 OpenAI、DeepSeek、硅基流动 (SiliconFlow) 等主流供应商。
3. 选择 **模型名称**：推荐使用 `deepseek-v3` 或 `gpt-4o-mini` 以平衡成本与质量。
4. 填写 **Base URL**：如果使用代理或国内镜像源（如硅基流动），请务必填写对应的接口地址。

### 第二步：建立信号链路（RSS 配置）
1. 进入 **[Insight 洞察]** 页面。
2. 点击左侧控制塔中 **[SIGNAL_SOURCES 信号源]** 旁的设置图标。
3. **添加订阅**：输入你关注的 RSS 链接（如 Nature, IEEE 等）。
4. **设定频率**：选择“每天 (9 AM)”或“每周一 (9 AM)”自动抓取。

### 第三步：配置交付终端（周报发送）
1. 在 **[Insight 洞察]** 页面左侧控制塔，找到 **[WEEKLY_PROTOCOL 周报协议]**。
2. 点击设置图标，在弹出窗口中填写 **[TARGET_EMAIL 接收邮箱]**。
3. 设定 **[SCHEDULE 频率]**：选择你希望收到邮件提醒的具体日期和时间（建议设置在 RSS 抓取时间之后，如周一 9:30 AM）。

## 高阶技巧：iOS 快捷指令 (Automation)

为了让你在手机上也能随时随地“一键捕捉”灵感，NeoFeed 支持 iOS 快捷指令集成：

1. **获取 API 密钥**：在 [设置页] 找到你的个人 API Key（以 `sk_neofeed_` 开头）。
2. **创建快捷指令**：
   - 打开 iOS “快捷指令” App，创建新指令。
   - 添加“获取 URL 内容”操作。
   - **URL**：设置为 `https://neofeed.cn/api/process-url`（根据实际部署地址调整）。
   - **方法**：POST。
   - **Headers**：添加 `Authorization: Bearer YOUR_API_KEY`。
   - **Request Body (JSON)**：包含 `url: [快捷指令输入]`。
3. **使用方式**：在 Safari 浏览器看到好文章时，点击“分享”按钮 -> 选择你创建的快捷指令，文章便会自动注入 NeoFeed 矩阵进行 AI 处理。
