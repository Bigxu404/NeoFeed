## 🎮 快速测试

### 1. 保存文本

在前端界面：
1. 输入标题（可选）
2. 粘贴任意文本
3. 点击「保存」

### 2. 保存链接

1. 复制任意网页 URL
2. 粘贴到内容框
3. 系统自动抓取内容
4. 点击「保存」

### 3. 启用 AI 处理

1. 勾选「启用 AI 处理」
2. 需要配置 `OPENAI_API_KEY`
3. 系统自动生成：
   - 摘要
   - 分类
   - 关键词

### 4. 使用 API

```bash
# 保存文本
curl -X POST http://localhost:8000/api/items \\
  -H "Content-Type: application/json" \\
  -d '{
    "content": "这是一条测试信息",
    "title": "测试标题",
    "enable_ai": false
  }'

# 获取列表
curl http://localhost:8000/api/items?limit=10

# 查看统计
curl http://localhost:8000/api/stats
```

## ⚙️ 配置（可选）

### 启用 AI 处理

1. 创建 `.env` 文件：
```bash
cp .env.example .env
```

2. 编辑 `.env`，添加：
```env
OPENAI_API_KEY=sk-your-key-here
ENABLE_AI_PROCESSING=true
```

3. 重启后端服务

### 自定义端口

**后端**:
```bash
python3 -m uvicorn api.main:app --port 8888
```

**前端**:
```bash
# 编辑 web/.env.local
NEXT_PUBLIC_API_URL=http://localhost:8888
```

## 🐛 故障排查

### 前端无法连接后端

**检查后端是否运行**:
```bash
curl http://localhost:8000/health
```

**预期响应**:
```json
{
  "status": "healthy",
  "database": "ok",
  "ai_enabled": false
}
```

### 数据库错误

**重新初始化**:
```bash
cd database
rm neofeed.db
python init_db.py
```

### 端口占用

**查看占用**:
```bash
lsof -i :8000  # 后端
lsof -i :3000  # 前端
```

**杀死进程**:
```bash
kill -9 <PID>
```
