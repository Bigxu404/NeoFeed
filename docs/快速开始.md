# NeoFeed 快速开始指南

## 🎯 30 秒启动

### 方式 1: 一键启动（推荐）

```bash
chmod +x start.sh
./start.sh
```

### 方式 2: 分步启动

**终端 1 - 后端**:
```bash
cd /Users/Zhuanz/NeoFeed
python -m uvicorn api.main:app --reload
```

**终端 2 - 前端**:
```bash
cd /Users/Zhuanz/NeoFeed/web
npm run dev
```

## 📋 首次使用

### 1. 安装依赖

```bash
# Python 依赖
pip install -r requirements.txt

# 前端依赖
cd web && npm install
```

### 2. 初始化数据库

```bash
cd database
python init_db.py
```

### 3. 配置环境（可选）

```bash
cp .env.example .env
# 编辑 .env，添加 OPENAI_API_KEY
```

## 🎮 基本使用

### 保存文本

1. 打开 `http://localhost:3000`
2. 在输入框粘贴文本
3. 点击「保存」

### 保存链接

1. 复制任意网页 URL
2. 粘贴到输入框
3. 系统自动抓取内容
4. 点击「保存」

### 启用 AI 处理

1. 勾选「启用 AI 处理」
2. 保存内容
3. 等待几秒，AI 自动生成：
   - 📝 3 句话摘要
   - 🏷️ 分类
   - 🔑 关键词

## 🔧 常见问题

### Q: 前端无法连接后端？

**A**: 检查后端是否启动：
```bash
curl http://localhost:8000/health
```

### Q: AI 处理不工作？

**A**: 两种可能：
1. 没有配置 `OPENAI_API_KEY`（正常，AI 可选）
2. 在 `.env` 中设置 `ENABLE_AI_PROCESSING=true`

### Q: 无法抓取 URL？

**A**: 检查：
1. URL 格式是否正确
2. 网络连接是否正常
3. Jina Reader API 是否可用

### Q: 数据保存在哪里？

**A**: 本地 SQLite 数据库：`database/neofeed.db`
