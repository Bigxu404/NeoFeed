## 二、工作区改动概览

### 1. 移动端 (Mobile)

- **新增 `web/app/mobile/`**
  - **landing**：移动端落地页
  - **login**：登录/注册页（含顶部与标题留白等样式优化）
  - **reader**：文章阅读页，支持笔记抽屉与 **Neo-AI** 半屏对话
  - **profile**：个人页
  - **settings**：设置页
  - **layout.tsx**：移动端布局
  - **page.tsx**：移动端首页
- **新增 `web/components/mobile/`**
  - 移动端通用组件：流列表、滑动卡片、导航、Feed 列表、鉴权守卫等
- **Neo-AI 卡片体验**
  - 卡片内顶部区域与下方文案/箭头间距收紧
  - 输入框与底部间距略减；聚焦输入框（键盘弹起）时，卡片与屏幕顶部保留留白，避免顶栏挤压

### 2. API 与定时任务

- **新增 `web/app/api/cron/`**
  - **extract-keywords**：关键词提取
  - **fetch-rss**：RSS 拉取
- **新增 `web/app/api/settings/`**
  - **rss**：RSS 相关设置接口

### 3. Dashboard 与发现

- **web/app/dashboard/**
  - **actions.ts**：仪表盘相关服务端逻辑调整
  - **discovery-actions.ts**：发现页逻辑增强
  - **feed-notes-actions.ts**（新）：Feed 笔记 CRUD
  - **reader-assistant-actions.ts**（新）：阅读页 Neo-AI 助手
- **web/components/dashboard/**
  - **CategorySection.tsx**：分类区块样式与交互
  - **DualPaneModal.tsx**：双栏弹窗
  - **Workbench.tsx**：工作台

### 4. AI 与数据

- **web/lib/ai.ts**：AI 能力扩展（摘要、分类、阅读助手等）
- **web/types/database.ts**：数据库类型与笔记等结构
- **web/inngest/functions/generate-weekly.ts**：周报生成逻辑微调

### 5. 页面与路由

- **web/app/page.tsx**：首页结构与入口调整
- **web/app/layout.tsx**：根布局（如 PWA / 移动端相关）

### 6. 登录与鉴权

- **web/app/login/actions.ts**：登录/注册 action 调整
